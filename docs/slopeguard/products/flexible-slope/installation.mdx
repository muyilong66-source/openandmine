# 安装方法

## 与传统工艺的关系

挂网喷射混凝土：原设计采用网喷土钉墙支护，其中放坡坡率 1:0.3，坡面注浆土钉间距 1.5m×1.5m，边坡的整体稳定性符合规范要求。

网喷构造：挂 φ6@200 双向钢筋网片，表面喷射 80±20mm 厚 C20 混凝土。

![图示](/img/slopeguard/flexible-slope/image1.png)

在不改变边坡放坡坡率和注浆土钉构造情况下，采用装配式面层体系取代传统网喷构造。坡面从上至下沿坡面铺设并固定装配式面层，面层具有定伸长负荷强度指标和防水性能。设计装配式面层 2% 定伸长负荷不小于 6KN/m；面层铺设完成后，在坡面纵横向布置 ∅8 钢丝绳、钢丝绳缠绕在锚钉外露端并拉紧，锚钉外露端用套筒固定。

坡顶翻边宽度不小于 0.8m，并采用 80mm 厚 C20 混凝土压顶，平台和坡脚翻边宽度不小于 0.3m，均采用 1m 长直径 14mm 钢筋固定；水平方向绿色装配式面层铺设应从一侧顺序铺设，搭接处与上口线垂直，水平搭接不小于 300mm。

开挖形成的边坡采用绿色装配式支护，插筋构造支护结构不变；采用绿色装配式面层产品取代网喷护面构造。

表面从上至下沿坡面铺设并固定装配式面层（GRF01），面层具有定伸长负荷强度指标和防水性能。设计装配式面层 2% 定伸长负荷不小于 6KN/m；面层铺设完成后，在坡面纵横向布置 ∅6 钢丝绳、钢丝绳缠绕在锚钉外露端并拉紧，锚钉外露端用卡扣固定。

坡顶翻边宽度不小于 0.8m，并采用 80mm 厚 C20 混凝土压顶，平台和坡脚翻边宽度不小于 0.3m，均采用 1m 长直径 18mm 钢筋固定；水平方向绿色装配式面层铺设应从一侧顺序铺设，搭接处与上口线垂直，水平搭接不小于 200mm。
