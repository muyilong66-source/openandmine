# 技术参数

## 网喷构造（对照）

- 钢筋网：φ6@200 双向网片  
- 喷射混凝土：80±20mm 厚 C20  

## 装配式面层（GRF01）设计指标

- 2% 定伸长负荷：不小于 **6 kN/m**  
- 坡顶翻边宽度：不小于 **0.8 m**，压顶 **80 mm** 厚 C20 混凝土  
- 平台、坡脚翻边宽度：不小于 **0.3 m**  
- 纵横向拉结：∅8 或 ∅6 钢丝绳（与锚钉外露端拉紧、套筒或卡扣固定）  
- 水平搭接：网喷替代方案 **≥300 mm**；GRF01 方案 **≥200 mm**  
- 坡脚/翻边锚固筋：**1 m** 长，直径 **14 mm** 或 **18 mm**（按构造选用）  

## 施工效率（对照）

| 项目     | 传统工艺（网喷类）     | 装配式面层           |
| -------- | ---------------------- | -------------------- |
| 效率     | 制约因素多，约 100 平/d | 装配化安装，约 500 平/d |
| 工种要求 | 需专业工种             | 无需专业工种，简单机械即可 |

## 绿色施工（对照）

- 传统：需空压机、喷锚机、挖掘机（干喷上料）或混凝土罐车（湿喷上料）；速凝剂、水、电等消耗。  
- 装配式：无需动力设备为主，自然放坡 GRF01 装配以轻型手持式设备为主；总体施工阶段**碳减排约 60%**（文内表述）。
