# 原理

## 安全性对比

支护体系的土钉群受力体系，保证边坡稳定性。在整体稳定性满足安全条件下，网喷主要功能为：（1）承担土钉间土压力变化；（2）防止外部水侵蚀坡面；（3）防止内部水冲刷坡面。

混凝土面层现作，不宜控制质量，其变形与坡面土体不一致，坡体发生不均匀变形后，易导致裂缝，混凝土面层功能性受损，坡面易被冲蚀。

支护体系的土钉群受力原理等同，保证边坡稳定性。在整体稳定性满足安全条件下，装配式面层主要功能为：（1）承担土钉间土压力变化；（2）防止外部水侵蚀坡面；（3）防止内部水冲刷坡面；（4）及时有效地完成防护，尤其是雨季。

土体发生变形后装配式面层与土体变形协同，但面层不会破坏，保证功能性完整。

## 工艺优缺点

**传统工艺（喷射混凝土）**

（1）常规喷射混凝土施工时有噪音、废气、扬尘等多个污染源。针对临时结构而言，喷射混凝土施工工艺在当前环保政策要求下已较为落后。  
（2）材料散放，二次转运困难，喷射混凝土施工，浪费严重。  
（3）受天气影响，雨季施工影响大，施工效率低。  

**绿色装配式面层**

（1）使用材料绿色环保、轻型、美观。  
（2）施工过程无污染，绿色施工。  
（3）产品为工厂定制、安装简单，功效快。使用小型机具即可安装，人均产能提高 2 倍，且受天气影响小、不受材料供给环境限制，支护分项工期较传统可节约 3～5 倍。  
（4）面层产品造价与传统相当。  

## 护坡质量

- **网喷**：厚度浮动较大，受施工工人技术水平影响大，面层厚度不稳定。  
- **装配式**：工业化集约生产面层，指标稳定，质量有保证。
